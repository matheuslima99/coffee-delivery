<!-- GETTING STARTED -->
## Começando

Você precisa ter o NodeJs instalado



###   Instalação

1. Clonar o repositório
   ```sh
   git clone https://github.com/matheuslima99/coffee-delivery
   ```
2. Instalar pacotes NPM
   ```sh
   npm install
   ```
3. Para executar no seu dispositivo

   ```sh
   npm run dev 
   ```
